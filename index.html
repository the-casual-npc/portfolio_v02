Game of Life

While learning python at school, we got assigned multiple projects, including this one. It is a Python recreation of the Convey's Game of Life.

The project was made using PyCharm, and in the game, you can choose the size of the game board, the percentage starting with live cells (the placement is random), and the number of iterations the game runs for. The game is using ASCII art for its design.

The Maze Game

This is another python school project. Once you start the game you can move around the maze using W, A, S, D. The goal is to get to the exit while avoiding traps and enemies.

The maze is randomly generated each time, with selectable size. There are toggleable enemies, traps, healing potions and fog of war. It was also made in PyCharm. The game can be saved and loaded anytime. The menus are designed with a combination of ASCII art and emojis.
